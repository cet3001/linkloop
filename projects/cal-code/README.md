# Cal Code

Cal Code is a local-first AI development environment built for personal software engineering workflows.

## Principles

- Local-first by default (Ollama runtime, no cloud APIs required)
- Modular architecture with clean system boundaries
- Replaceable components for models, agents, memory, and tools
- Desktop-first UX powered by Tauri + React + Monaco

## Structure

- `apps/` runtime systems and product surfaces
- `packages/` shared domain libraries
- `docs/` architecture and delivery documentation
- `scripts/` developer and automation scripts
- `configs/` centralized configuration files

## AI Architecture

Cal Code uses a local-first architecture with Ollama runtime, model routing, context ranking, and tool-aware execution loops.
